# 微博热榜监控

微博热搜榜实时监控，支持热度分析、趋势追踪、舆情监控。

## 功能

### 热榜获取
- **实时热搜** - 获取当前微博热搜榜 Top 50
- **热度指数** - 显示每条热搜的热度值
- **标签分类** - 新、热、爆等标签

### 趋势分析
- **上升最快** - 热度上升最快的话题
- **持续热度** - 长期热门话题追踪
- **地域分布** - 热搜地域分布（如可获取）

### 舆情监控
- **关键词监控** - 监控特定关键词上热搜
- **提醒功能** - 热搜变化时主动提醒
- **历史记录** - 保存热搜历史快照

## 使用方式

### 获取热搜榜

```
获取微博热搜榜前 10
```

返回：
```json
[
  {"rank": 1, "title": "春节档电影票房", "hot": 9876543, "label": "爆"},
  {"rank": 2, "title": "2026春运", "hot": 8765432, "label": "热"},
  {"rank": 3, "title": "AI技术突破", "hot": 7654321, "label": "新"}
]
```

### 关键词监控

```
监控微博热搜中的 "AI" 相关话题
```

### 趋势分析

```
分析微博热搜今天的趋势变化
```

## 数据来源

- 微博热搜 API（如可用）
- 模拟数据（API 不可用时）
- 历史快照对比

## 注意事项

1. **API 限制** - 微博 API 可能需要授权或有频率限制
2. **数据时效** - 热搜榜每 5 分钟更新一次
3. **地域差异** - 不同地区热搜可能有差异
4. **合规使用** - 遵守微博平台规则
