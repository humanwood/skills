# USDC Krump (USDC.k) EVVM Payment Skill for OpenClaw

Enable autonomous agent payments with USDC Krump (USDC.k) via x402 protocol on Story Aeneid EVVM.

## Installation

```bash
clawhub install usdc-dance-evvm-payment
```

Or clone this repository:

```bash
git clone https://github.com/arunnadarasa/usdcdance.git
cd usdcdance/openclaw-skill-usdc-dance-evvm
npm install
```

## Usage

See `SKILL.md` for complete documentation and `examples/` for usage examples.

## Publishing to ClawHub

1. Ensure all files are in place
2. Test the skill locally
3. Publish to ClawHub:

```bash
clawhub publish
```
