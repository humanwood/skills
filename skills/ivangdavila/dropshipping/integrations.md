# E-commerce Integrations Reference

## Store Platforms

### Shopify
- **API**: REST Admin API + GraphQL Admin API
- **Key features**: products, orders, fulfillment, webhooks, themes (Liquid)
- **Fees**: Basic $29, Shopify $79, Advanced $299/month + transaction fees
- **Dropshipping apps**: DSers (official AliExpress partner), AutoDS, Spocket

### WooCommerce
- **API**: REST API v3
- **Key features**: self-hosted (WordPress), unlimited customization, plugin ecosystem
- **Costs**: hosting + premium plugins
- **Dropshipping plugins**: AliDropship, WooDropship, Dropified

### BigCommerce
- **API**: REST + GraphQL Storefront
- **Key features**: multi-storefront native, headless capabilities, B2B features
- **Pricing**: based on annual sales

## Marketplaces

### Amazon
- **API**: SP-API (Selling Partner API)
- **Dropshipping rules**: allowed but supplier must remove their branding
- **Fees**: referral 8-15% + FBA fees if using fulfillment
- **Critical metrics**: ODR <1%, late shipment <4%

### eBay
- **APIs**: Browse API, Sell APIs, Fulfillment API
- **Dropshipping**: allowed with restrictions
- **Fees**: insertion + final value 12-15%

### Etsy
- **API**: Open API v3
- **Best for**: handmade, vintage, craft supplies, print-on-demand
- **Fees**: $0.20 listing + 6.5% transaction

## Suppliers/Sourcing

### AliExpress
- **Integration**: via DSers, AutoDS, or direct API
- **Shipping**: ePacket 15-30 days, AliExpress Standard
- **Tip**: AliExpress Dropshipping Center for product research

### CJ Dropshipping
- **API**: CJ API
- **Advantages**: US/EU warehouses, POD included, branding options
- **Better shipping**: faster than AliExpress direct

### Spocket
- **Integration**: Shopify, WooCommerce, Wix, BigCommerce
- **Focus**: US/EU suppliers (faster shipping)
- **Pricing**: Free, Starter $24, Pro $49/month

### Print-on-Demand
- **Printful**: leader, global warehouses, branding options
- **Printify**: network of print providers, compare prices
- **Gooten**: API-first, good for high volume

## Automation Tools

### Order Processing
- **DSers**: official AliExpress partner, bulk orders, supplier optimizer
- **AutoDS**: full automation, 25+ suppliers, price monitoring
- **Dropified**: similar features, Chrome extension

### Shipping
- **ShipStation**: multi-carrier, automation rules, branded tracking
- **Easyship**: courier comparison, duties calculator
- **AfterShip**: tracking automation, 1100+ carriers, branded tracking page

## Payments

### Primary Processors
- **Stripe**: 2.9% + $0.30, fraud detection (Radar), multi-currency
- **PayPal**: 2.9% + $0.30, buyer protection (chargeback risk), 21-day holds for new sellers
- **Shopify Payments**: Stripe-powered, no additional transaction fees

### BNPL (Buy Now Pay Later)
- **Klarna**: increases conversion 20-30%, fees 4-6%
- **Afterpay/Clearpay**: 4 payments, fees 4-6% + fixed

### Regional
- **LATAM**: Mercado Pago, PagSeguro, OXXO (Mexico)
- **Europe**: SEPA, iDEAL (NL), Bancontact (BE), Sofort
- **Asia**: Alipay, WeChat Pay

## Marketing

### Advertising
- **Meta Ads**: Marketing API, Conversions API, Pixel + CAPI for iOS tracking
- **Google Ads**: Shopping, Search, Performance Max, Merchant Center setup
- **TikTok Ads**: TikTok Pixel, Spark Ads (boost organic), younger demographic

### Email Marketing
- **Klaviyo**: e-commerce focused, Shopify native, flows (abandoned cart, post-purchase)
- **Omnisend**: email + SMS + push, pre-built automations
- **Mailchimp**: more general, generous free tier

### SMS
- **Postscript**: Shopify-focused
- **SMSBump**: affordable option
- **Note**: TCPA compliance required in US

## Analytics

### Traffic & Conversion
- **Google Analytics 4**: event-based tracking, enhanced e-commerce
- **Google Tag Manager**: centralize all tracking tags

### Attribution
- **Triple Whale**: first-party data, profit tracking, creative analytics
- **Northbeam**: multi-touch attribution, media mix modeling

### Profit Tracking
- **BeProfit**: profit analytics, COGS tracking
- **Lifetimely**: LTV predictions, cohort analysis
- **OrderMetrics**: real-time profit, multi-store

## Customer Service

### Helpdesks
- **Gorgias**: e-commerce focused, revenue tracking per ticket
- **Zendesk**: enterprise-grade, multi-channel
- **Freshdesk**: affordable, free tier available

### Reviews
- **Judge.me**: affordable, photo/video reviews, free plan
- **Loox**: photo reviews, import from AliExpress
- **Yotpo**: reviews + UGC + loyalty (enterprise)
