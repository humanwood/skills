# Student Guide — Paris

## Overview

Paris is a top student destination globally. Affordable tuition (even for internationals), world-class universities, and vibrant student life. Main challenge: finding affordable housing.

## Housing Options

| Type | Price | Where to Find |
|------|-------|---------------|
| **CROUS residence** | €200-600 | crous-paris.fr (apply early!) |
| **Shared apartment** | €500-800 | La Carte des Colocs, Appartager |
| **Studio** | €800-1,200 | PAP.fr, Seloger, LeBonCoin |
| **Homestay** | €600-900 | Cohome, Cohabit |

**Tips:**
- Apply for CROUS housing in April-May for September
- CAF housing aid: €100-250/month (apply immediately)
- Guarantor often required (Visale = free government guarantee)

## Visa (Campus France)

### Process for Non-EU

1. **Apply online** via Campus France in home country
2. **University acceptance** letter required
3. **Campus France interview** (in home country)
4. **Visa application** at French consulate
5. **OFII validation** upon arrival in France

| Document | Notes |
|----------|-------|
| Acceptance letter | Original from university |
| Financial proof | €615/month minimum (~€7,400/year) |
| Housing proof | Address in France |
| Insurance | Required |

**Timeline:** Start 6+ months before arrival
