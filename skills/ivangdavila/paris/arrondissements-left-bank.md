# Left Bank Arrondissements (5-7) — Paris

## 5th Arrondissement (Latin Quarter)

| Aspect | Details |
|--------|---------|
| **Vibe** | Academic, student, historic |
| **Landmarks** | Panthéon, Luxembourg Gardens, Sorbonne |
| **Rent (1BR)** | €1,600-2,200 |
| **Safety** | ★★★★☆ |
| **Best For** | Students, academics, intellectual types |
| **Metro** | 7, 10, RER B |

**Pros:** Charming streets, bookshops, affordable-ish for central
**Cons:** Tourist trap restaurants, student chaos

| Aspect | Details |

| **Rent (1BR)** | €2,500-3,500 |

## 7th Arrondissement (Eiffel Tower)

| Aspect | Details |
|--------|---------|
| **Vibe** | Diplomatic, quiet, upscale |
| **Landmarks** | Eiffel Tower, Musée d'Orsay, Invalides |
| **Rent (1BR)** | €2,200-3,200 |
| **Safety** | ★★★★★ |
| **Best For** | Families, expats, professionals seeking quiet |
| **Metro** | 8, 13, RER C |

**Pros:** Beautiful, safe, Rue Cler market
**Cons:** Expensive, quiet at night, not hip

## Left Bank Character

The "Rive Gauche" has traditionally been:
- More intellectual (universities)
- More artistic (galleries)
- More literary (publishing houses)
- Quieter than Right Bank

### Best Streets

| Street | What's There |
|--------|--------------|
| **Rue de Buci** | Market street, cafés |
| **Rue Cler** | Best market street |
| **Rue Mouffetard** | Historic, touristy but charming |
| **Boulevard Saint-Germain** | Cafés, shops |
