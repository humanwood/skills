# Food Practical Guide — Paris

## Meal Times

| Meal | Typical Hours | Notes |
|------|---------------|-------|
| **Breakfast** | 7:00-10:00 | Coffee + croissant only |
| **Lunch** | 12:00-14:00 | Main meal for many; fixed menu deals |
| **Goûter** | 16:00-17:00 | Afternoon snack (pastry) |
| **Apéro** | 18:00-20:00 | Pre-dinner drinks |
| **Dinner** | 20:00-22:00 | Later than Anglo countries |

**Critical:** Kitchens close between meals. Don't expect food at 15:00 or 18:00.

---

## Reservations

| Restaurant Type | Need Booking? |
|-----------------|---------------|
| Trendy/popular | Yes, 1-4 weeks ahead |
| Michelin starred | Yes, 1-2 months ahead |
| Neighborhood bistros | Usually no |
| Cafés | Never |
| Lunch anywhere | Usually no |

**Booking platforms:** LaFourchette (TheFork), OpenTable, restaurant websites.

---

## Tipping

| Situation | Tip |
|-----------|-----|
| Restaurant | Service included; round up or €2-5 for good service |
| Café | €0.50-1 on counter |
| Taxi | Round up |
| Hotel | €1-2 per bag |
| Hairdresser | 10% |

**Note:** "Service compris" means tip is included. Tipping is appreciated but never expected.

## Budget Eating

| Strategy | How |
|----------|-----|
| **Lunch formule** | 2-3 course set menu, €15-20 |
| **Bakery sandwich** | €5-8, quality ingredients |
| **Market picnic** | Cheese + bread + fruit = €10 |
| **Supermarket prepared** | €5-10 meals |
| **Bouillon restaurants** | Traditional, €15-20 for full meal |
| **Happy hour** | 17:00-20:00, 50% off drinks |
| **Carafe d'eau** | Free tap water (ask for it) |

## Common Mistakes

| Mistake | Reality |
|---------|---------|
| Ordering "entrée" thinking main | Entrée = starter |
| Expecting free bread refills | Usually yes, but not always |
| Asking for tap water wrong | Say "carafe d'eau, s'il vous plaît" |
| Rushing meal | French dine slowly; enjoy |
| Not checking closing times | Kitchens close early |
| Touristy restaurants | If menu has photos, leave |
