📦 导入 Skill
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
cd ~/.claude/skills
tar -xzf video_analyzer-skill-v2.0.0.tar.gz
cd video_analyzer
pip install -r requirements.txt

⚙️ 配置 API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
cp config.example.json config.json
# 编辑 config.json，填入 API key

🚀 快速使用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# 基础用法（自动启用截图）
python run.py --url "视频链接"

# 快速模式（使用小模型）
python run.py --url "视频链接" --whisper-model small

# 禁用截图（更快）
python run.py --url "视频链接" --no-screenshots

📊 模型选择
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
tiny      - 75MB   | 最快 | 低精度  | 快速测试
small     - 461MB  | 快   | 中精度  | 推荐日常使用 ⭐
medium    - 1.5GB  | 中   | 高精度  | 平衡选择
large-v2  - 2.87GB | 慢   | 最高    | 正式使用

⚠️ 系统要求
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ Python 3.8+
✓ FFmpeg（必需）
  Windows: winget install ffmpeg
  macOS:   brew install ffmpeg
  Linux:   sudo apt install ffmpeg
✓ LLM API key（OpenAI/Anthropic/其他）

🐛 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Q: 提示缺少 FFmpeg
A: 安装 FFmpeg（见上方系统要求）

Q: API 调用失败
A: 检查 config.json 中的 API key

Q: 首次运行很慢
A: 正在下载 Whisper 模型（仅首次需要）

Q: 视频下载失败
A: 检查网络连接或使用本地视频文件
