# 项目类型完整说明

## 内置类型

### 💻 代码项目
**适用场景**:
- Web 应用、移动应用开发
- 工具脚本、自动化程序
- 开源项目、个人作品
- API 服务、后端系统

**示例**:
- 客服话术对练系统
- Gemini Agent Client
- 个人博客网站
- 数据爬虫脚本

---

### 📚 学习计划
**适用场景**:
- 职业资格证书（PMP、CPA 等）
- 语言学习（英语、日语等）
- 技术学习（Python、AI 等）
- 在线课程、培训班

**示例**:
- PMP 备考（目标 3 个月）
- 雅思学习（目标 7.0）
- Python 入门（30 天完成）

---

### 💼 工作目标
**适用场景**:
- OKR、KPI 目标
- 季度/年度绩效
- 晋升准备
- 重点项目交付

**示例**:
- Q1 OKR - 用户增长 30%
- 年度绩效 - S 级评估
- 晋升准备 - 高级工程师

---

### 🏠 个人事务
**适用场景**:
- 搬家、装修
- 购车、购房
- 家庭计划
- 证件办理

**示例**:
- 新房装修（3 月完工）
- 购车计划（预算 20 万）
- 落户办理

---

### 🎨 创意项目
**适用场景**:
- 写作（小说、文章）
- 设计（UI、平面）
- 视频制作
- 艺术创作

**示例**:
- 科幻小说创作（10 万字）
- 个人 IP 设计
- YouTube 频道运营

---

### 💪 健康管理
**适用场景**:
- 健身计划
- 减重/增肌
- 饮食管理
- 体检、医疗

**示例**:
- 减重计划（目标 -10kg）
- 健身打卡（每周 3 次）
- 年度体检安排

---

### 💰 财务规划
**适用场景**:
- 投资理财
- 预算管理
- 储蓄计划
- 保险配置

**示例**:
- 基金定投计划
- 买房储蓄计划（目标 100 万）
- 年度预算管理

---

### ✈️ 旅行计划
**适用场景**:
- 国内/出境游
- 商务出差
- 周末短途
- 探亲访友

**示例**:
- 日本旅行（5 月黄金周）
- 春节回家
- 周末莫干山短途

---

## 自定义类型

当内置类型不适用时，可以创建新类型：

**格式**:
```
| 类型名称 | Emoji | 说明 |
```

**示例**:
```
用户：我想管理养猫相关的事情
→ 添加：| 宠物养护 | 🐱 | 养宠相关事项 |
```

**建议**:
- Emoji 选择要直观
- 说明简洁明确
- 避免类型过多（建议不超过 15 种）
