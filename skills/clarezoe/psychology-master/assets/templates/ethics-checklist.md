# Ethics Checklist Template

## Transparency
- [ ] Claims are clear and verifiable.
- [ ] No hidden conditions in key messaging.
- [ ] Limitations are visible before commitment.

## Autonomy
- [ ] No dark patterns or forced consent.
- [ ] User can opt out without penalty.
- [ ] Defaults do not exploit inattention.

## Vulnerable Groups
- [ ] Extra caution for minors or sensitive audiences.
- [ ] Language avoids shame, fear, or manipulation.
- [ ] Escalation path exists for harmful outcomes.

## Data Respect
- [ ] Data requested is necessary and minimal.
- [ ] Retention period is stated.
- [ ] Access and deletion flow is documented.

## Experiment Safety
- [ ] Success metrics include user well-being signals.
- [ ] Guardrails are defined before launch.
- [ ] Stop conditions are explicit.
