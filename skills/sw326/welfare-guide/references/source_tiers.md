# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **복지로** (bokjiro.go.kr) — 중앙부처·지자체 복지서비스 공식 포털
- **보조금24** (gov.kr/정부24) — 정부 수혜서비스 공식 포털
- **보건복지부** (mohw.go.kr) — 복지 정책 원천
- **국민건강보험공단** (nhis.or.kr) — 노인장기요양 기준
- **주민센터/행정복지센터** — 지자체 지원 최신 정보

## Tier 2 — 높은 신뢰도
- **복지로 공식 블로그** — 제도 설명 콘텐츠
- **보건복지부 보도자료** — 정책 변경 최신
- **아이사랑** (childcare.go.kr) — 보육 관련 공식 정보

## Tier 3 — 보강용
- 각 지자체 공식 홈페이지 — 지역별 추가 지원 확인

## 금지 출처
- 개인 블로그·카페·인플루언서 지원금 정보 → 인용 금지 (오정보 많음)
- 지원금 신청 대행 광고성 콘텐츠 → 인용 금지

## 인용 형식
```markdown
# API
(보조금24 API: 서비스명)
(복지로 중앙부처 API: 서비스ID)
(복지로 지자체 API: 지역코드)

# 웹
[복지로](https://www.bokjiro.go.kr) — 서비스명
[보건복지부](https://www.mohw.go.kr) — 정책명

# 지자체 추가 지원 (변동 가능)
"[지역] 지자체 추가 지원 — 변동 가능, 주민센터 확인 권장"
```

## ⚠️ 복지 정보 특별 규칙
- **정책 변동 명시**: 복지 정책은 자주 바뀜 → "YYYY년 기준" 명시 필수
- **지자체 차이 명시**: 지역별 추가 지원 있을 수 있음 → 주민센터 확인 권고
- **수급 확정 금지**: "받을 수 있을 것 같습니다" ← "받을 수 있습니다" ❌
