# 인텐트 라우터 (9개 모드)

> 공통 분류 규칙: [`../../common/intent_router_base.md`](../../common/intent_router_base.md)

## 분류 규칙 (공통)
사용자 요청을 아래 9개 인텐트 중 하나로 분류한다.
복합 요청은 주 인텐트 + 보조 인텐트로 처리한다.
파라미터 부족 시 1회 질문. 추측 가능하면 추측 후 명시.

---

## 인텐트 목록

### 1. Benefit Search (맞춤 혜택 조회)
- **표현:** "나 받을 수 있는 혜택 뭐야?", "지원금 뭐 있어?", "정부지원 알려줘"
- **산출물:** 입력 조건 기반 맞춤 혜택 목록 (중앙 + 지역 통합)
- **최소 데이터:** 없음 (나이/지역/가구 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시
- **조건 수집:** 나이, 지역, 가구원 수, 소득 수준, 특수 상황(임신/장애/실직 등)

### 2. Childcare (육아/보육)
- **표현:** "아이 키우면 지원금 뭐 있어?", "보육료 지원", "아동수당", "어린이집 비용"
- **산출물:** 자녀 연령별 육아 지원 패키지 Flash
- **최소 데이터:** 자녀 나이 (없으면 0~7세 전체 안내)
- **깊이:** Flash + Deep-Dive 요청 시

### 3. Birth Support (출산 지원)
- **표현:** "출산하면 뭐 받아?", "임신 지원금", "첫만남이용권", "출생축하금", "산전검사 지원"
- **산출물:** 임신·출산·신생아 단계별 지원 총정리
- **최소 데이터:** 없음 (지역 있으면 지자체 지원 추가)
- **깊이:** Flash + Deep-Dive 요청 시
- **중요:** 지자체별 추가 지원금 차이 명시

### 4. Basic Living (기초생활수급)
- **표현:** "기초생활수급자 되려면?", "생계급여 얼마야?", "의료급여", "차상위계층"
- **산출물:** 급여 종류별 선정 기준 + 급여액 + 신청 방법
- **최소 데이터:** 없음 (가구/소득 있으면 해당 여부 추정)
- **깊이:** Flash + Deep-Dive 요청 시

### 5. Youth Support (청년 지원)
- **표현:** "청년 혜택 뭐 있어?", "청년수당", "청년도약계좌", "취업성공패키지", "청년 주거"
- **산출물:** 나이별 청년 전용 지원 프로그램 목록
- **최소 데이터:** 없음 (나이 있으면 정확한 대상 여부 확인)
- **깊이:** Flash + Deep-Dive 요청 시

### 6. Senior Support (노인 복지)
- **표현:** "부모님 노인 지원금", "기초연금", "노인장기요양", "경로우대", "노인 일자리"
- **산출물:** 나이별 노인 복지 서비스 + 수급 조건
- **최소 데이터:** 없음 (나이 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시

### 7. Disability Support (장애인 지원)
- **표현:** "장애인 지원금", "장애수당", "장애인연금", "활동지원서비스", "장애아동수당"
- **산출물:** 장애 유형·등급별 복지 서비스 목록
- **최소 데이터:** 없음 (장애 등급/유형 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시

### 8. How to Apply (신청 방법)
- **표현:** "어떻게 신청해?", "온라인 신청 가능해?", "서류 뭐 필요해?", "주민센터 가야 해?"
- **산출물:** 신청 경로 (복지로/정부24/주민센터) + 필요 서류 목록
- **최소 데이터:** 지원 종류
- **깊이:** Flash only

### 9. Beginner Guide (입문)
- **표현:** "복지가 뭐야?", "지원금 처음인데", "어디서 신청해?", "복지로가 뭐야?"
- **산출물:** 복지 시스템 입문 Flash (초보자 모드 강제)
- **최소 데이터:** 없음
- **깊이:** Flash only

## 공통 인텐트

### 🔰 Beginner Mode 자동 전환
키워드: "쉽게", "기초", "처음인데", "어떻게 해?"

### ❓ Scope Out
복지 외 질문 → 안내
예: "연금 계산" → CHM-239 연금/노후 스킬 안내 (연금 외 노인 복지는 커버)
예: "창업 지원금" → CHM-240 창업/사업자 스킬 안내 (단, 복지 성격이면 커버)
