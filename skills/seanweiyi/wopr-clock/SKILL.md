# W.O.P.R. Clock (TACO Tracker)

This is a retro-styled terminal clock inspired by the W.O.P.R. computer from the movie WarGames. It tracks the countdown to the TACO event on March 6th, 2026.

## Features

- **Retro Aesthetic**: Phosphor green text on a black background with CRT flicker effects.
- **Cross-Platform**: Optimized for both Windows and MacOS.
- **Always-on-top**: Keeps the mission-critical countdown visible at all times.

## Usage

### Launch Clock
```bash
python scripts/wopr_universal.py
```
